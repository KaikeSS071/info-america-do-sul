# Aplicação de Pesquisa de Países - imeerçao dev alura com google gemini.

## Descrição
Esta aplicação web permite aos usuários pesquisarem por países e obterem informações básicas sobre cada um, como:

* **Nome:** O nome oficial do país.
* **Descrição:** Uma breve descrição sobre a história, cultura e geografia do país.
* **Idioma:** O idioma oficial ou os idiomas mais falados.
* **Moeda:** A moeda utilizada no país.
* **População:** A estimativa da população total.
* **Bandeira:** Um link para a imagem da bandeira nacional.
* **Mais informações:** Um link para a página da Wikipédia do país.

## Tecnologias Utilizadas
* **HTML:** Estrutura básica das páginas.
* **CSS:** Estilização visual da aplicação.
* **JavaScript:** Lógica para a pesquisa e exibição dos resultados.

## Como Funciona
1. **Entrada do usuário:** O usuário digita o nome do país desejado em um campo de pesquisa.
2. **Processamento:** O JavaScript busca na lista de países pré-definida por uma correspondência com a entrada do usuário.
3. **Saída:** Os resultados da pesquisa são exibidos em uma lista, com cada item contendo as informações relevantes sobre o país encontrado.

## Estrutura do Projeto
* **index.html:** Arquivo principal da página web.
* **style.css:** Arquivo de estilos CSS para personalizar a aparência da página.
* **app.js:** Arquivo JavaScript contendo a lógica da aplicação.
* **dados.js:** Arquivo JavaScript contendo os dados dos países (nome, descrição, etc.).
